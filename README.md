MDWriter is a modern, elegant, and powerful Markdown editor for macOS, built with SwiftUI and SwiftData. Inspired by professional writing tools like Ulysses, it provides a focused writing environment with powerful library management and export capabilities.

## Key Features

- **Focused Writing Experience**: A clean, distraction-free interface with typewriter mode.
- **Advanced Markdown Support**: Real-time syntax highlighting and intelligent Markdown parsing.
- **Powerful Library Management**: Organize your notes with folders and nested groups. SwiftData-powered for performance and reliability.
- **Document Outline**: Navigate through long documents easily with the integrated outline view.
- **Live Statistics**: Keep track of character count, word count, and estimated reading time.
- **Flexible Export**: Export your work to PDF, Rich Text (RTF), or standard Markdown.
- **Personalized Themes**: Support for Light and Dark modes, plus customizable editor themes (Pure, etc.).
- **Typography Control**: Fine-tune your writing environment with adjustable fonts, line height, line width, and indents.
- **Snapshots & Backups**: Never lose your work with built-in snapshot history and full library backup/restore (.mdwbk).
- **Modern Tech Stack**: Built entirely with SwiftUI, SwiftData, and the latest macOS APIs.

## Tech Stack

MDWriter leverages several powerful open-source libraries:
- **SwiftUI**: For a modern, native macOS user interface.
- **SwiftData**: For persistent storage and data management.
- **swift-markdown**: For high-performance Markdown parsing.
- **Highlightr**: For real-time code and syntax highlighting.
- **Sparkle**: For seamless app updates.
- **WhatsNewKit**: To showcase new features after updates.

## Getting Started

### Prerequisites
- macOS 14.0 or later
- Xcode 15.0 or later

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/MDWriter.git
   ```
2. Open `MDWriter.xcodeproj` in Xcode.
3. Build and run the project (`Cmd + R`).
